# Family-Banking-System-

## Project overview
This is a Family Bankking system developed using Java with its OOP principles.

This application is role based, which means:
- Parents can access, monitor, deposit money, view balance and approve or reject requests.
- Child can view balance, spend money and request money.

## Technologies Used
- Java
- JavaFX
- SQLite
- JDBC


## Features
### Parent Features
- Parent registration and login
- Link child accounts to parent account
- Deposit money to child account
- View child balance
- Approve or reject money requests

### Child Features
- Child registration and login
- View balance
- Spend money
- Request money

## Database 
This application uses **SQlite via JDBC**

## Requirements
- Java 17 or later
- JavaFX SDK

## Steps
1. Clone this repo
2. Open project
3. Configure JavaFX
4. Run "LoginFX.java"
